\# Inventory Management System



\## Project Overview



Inventory Management System is a real-time backend application developed using Python Flask, SQL Server and REST APIs.



The system manages products, customers, suppliers, users and provides sales analytics dashboard.



\---



\## Technologies Used



\- Python

\- Flask

\- SQL Server

\- pyodbc

\- REST API

\- JWT Authentication

\- Swagger API Documentation

\- HTML

\- JavaScript

\- Chart.js

\- Git \& GitHub



\---



\## Features



\### Product Management



\- Add Product

\- View Products

\- Update Product

\- Delete Product

\- Search Product





\### Customer Management



\- Add Customer

\- View Customer

\- Update Customer

\- Delete Customer





\### Supplier Management



\- Add Supplier

\- View Supplier

\- Update Supplier

\- Delete Supplier





\### User Management



\- User Registration

\- Login Authentication

\- Total Sales

Charts:



\- Category Sales

\- Top Products

\- Monthly Sales Trend
